# Quality Threshold Rationale and Calibration

## Current Thresholds (pre-optimization)
- SUSPICIOUS_SOLID: >95% identical pixels (raised from 80%)
- SUSPICIOUS_LOW_VARIANCE: stddev < 5.0 for textures > 16x16 (lowered from 8)
- SUSPICIOUS_EXTREME: mean luminance < 5.0 or > 250.0
- SUSPICIOUS_DIMS: width/height = 0, >4096
